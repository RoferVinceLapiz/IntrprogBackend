# IPT 2026 Backend

Node.js + MySQL Authentication API with JWT and Refresh Tokens.

## Live API
https://ipt-2026-backend-rollo.onrender.com/api-docs/

## Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your values
4. Run `npm start`

## Environment Variables
See `.env.example` for required variables.

## Tech Stack
- Node.js
- Express
- MySQL (Aiven)
- Sequelize
- JWT
- Mailtrap (email testing)
